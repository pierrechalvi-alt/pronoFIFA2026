Placez votre logo dans ce dossier avec le nom exact :

`worldcup-logo.png`

L'application le chargera automatiquement via `index.html`.

Formats conseillés :
- PNG (fond transparent recommandé)
- hauteur proche de 72px pour garder un rendu net dans le header

Si le fichier est absent, l'application bascule automatiquement sur une image de secours distante.
